#### Requirements

- PHP 7.2.5
- Install Git
- Install NodeJS 12.x >=
- Install Composer
- Install Xampp

## Install

    npm install
    npm run dev

    Create new database refer to env


### Get Started
    
    php artisan migrate
    php artisan serve
    
    
### For Dummy Data

    php artisan db:seed
